# >>> Hospital Operations Analytics Dashboard: #

An end-to-end Hospital Operations Analytics project developed using **SQL, Python, Power BI, and Excel** to analyze hospital operations, patient demographics, doctor performance, appointments, billing, and revenue. The project transforms raw operational data into interactive dashboards and actionable business insights to support data-driven decision-making.

---

# >>> Project Overview: #

Healthcare organizations generate large volumes of operational data across patients, doctors, appointments, billing, prescriptions, and payments. Without proper analysis, valuable insights remain hidden.

This project demonstrates a complete Data Analytics workflow by collecting data from a relational hospital database, performing data cleaning and exploratory data analysis (EDA), building an optimized data model in Power BI, and developing interactive dashboards for hospital management.

The solution enables administrators to monitor key performance indicators (KPIs), evaluate operational efficiency, and identify opportunities for improving patient care and resource utilization.

---

# >>> Project Objectives: #

- Analyze hospital operational performance.
- Understand patient demographics and registration trends.
- Evaluate doctor workload and consultation performance.
- Monitor appointment completion and no-show rates.
- Analyze hospital revenue and payment performance.
- Develop interactive dashboards for executive decision-making.
- Generate business insights using data analytics.

---

# >>>Dataset Description: #

The project is based on a relational hospital database consisting of "8 interconnected tables."

| Table | Description |
|--------|-------------|
| Patients | Patient demographic information |
| Doctors | Doctor details, specialization, and experience |
| Departments | Hospital department information |
| Appointments | Appointment schedules and status |
| Encounters | Consultation records, diagnoses, and consultation duration |
| Prescriptions | Prescribed medications and treatment details |
| Billing | Billing records generated after consultations |
| Payments | Payment information and payment status |

These tables were integrated through appropriate relationships to support cross-functional analysis in Power BI.
